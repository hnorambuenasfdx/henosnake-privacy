Política de Privacidad — HENOsnake
Última actualización: 27 de abril de 2026
HENOsnake ("la App") es desarrollada por Héctor Norambuena ("nosotros"). Esta política describe qué información se procesa cuando usas la App.
1. Datos que recopilamos
La App puede recopilar los siguientes datos solo si otorgas tu consentimiento explícito en la pantalla inicial:
Ubicación geográfica aproximada: a nivel de país y ciudad, derivada del GPS del dispositivo. No se almacenan coordenadas exactas.
Puntaje obtenido en partidas: número entero asociado a la fecha y ciudad.
Modelo de dispositivo: por ejemplo "iPhone 15".
Si rechazas el consentimiento, la App funciona normalmente y no se recopila ningún dato.
2. Cómo se almacenan los datos
Todos los datos se almacenan únicamente en tu dispositivo, en un archivo JSON dentro del sandbox de la aplicación (carpeta Documents). Estos datos:
Nunca se envían a servidores externos.
Nunca se comparten con terceros.
No se utilizan para publicidad.
No se utilizan para identificarte personalmente.
Se eliminan completamente al desinstalar la App.
3. Datos no recopilados
La App no recopila:
Nombre, email, teléfono u otros datos personales (excepto un alias opcional de hasta 3 caracteres para el ranking local).
Identificadores de publicidad (IDFA).
Datos biométricos.
Contactos, calendario, fotos.
Historial de navegación.
4. Permisos del sistema
Permiso	Uso
Ubicación (mientras se usa)	Determinar país y ciudad para estadísticas locales agregadas. Solicitada solo si aceptas el consentimiento inicial.
Puedes revocar este permiso en cualquier momento desde Ajustes → HENOsnake → Ubicación.
5. Menores de edad
La App está clasificada para usuarios de 4 años o más. No recopilamos datos personales de niños. El consentimiento de ubicación debería ser otorgado por un adulto responsable.
6. Tus derechos
Como usuario tienes derecho a:
Rechazar la recolección de datos en cualquier momento desinstalando la App o revocando permisos.
Acceder a los datos almacenados localmente (archivo henosnake_analytics.json en el contenedor de la App).
Eliminar todos los datos desinstalando la App.
7. Cambios en esta política
Si cambia esta política, se publicará una nueva versión en esta misma URL. Los cambios sustanciales se comunicarán a través de una actualización de la App.
